# Rhino2Abaqus-GH
在Grasshopper中，将复杂钢节点的Rhino模型转化为Abaqus模型 (包含几何信息和材料信息)。

Transform Rhino model of complex steel joints into Abaqus model in Grasshopper (including geometric information and material information).
